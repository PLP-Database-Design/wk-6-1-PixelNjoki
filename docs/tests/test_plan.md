# Test Plan — Book Store App

## Objective and Scope
the objective is to test the Book Store web app to make sure users can browse, buy, and manage books properly.

## In-Scope Features (mapped to FR codes)
- FR1: User Login/Signup
- FR2: Search for Books
- FR3: Add to Cart
- FR4: Checkout and Payment
- FR5: Order History

## Out-of-Scope
- Admin dashboard (not part of user testing)
- Mobile version

## Test Environment
- Browser: Chrome
- OS: Windows 11
- Server: Localhost
